Image Segmentation
===================

Simple test with flask fro process of partitioning a digital image into multiple segments

### Requirements ###
- python 3.x
- pip
- virtualenv

### Installation ###
```shell
  virtualenv env
  source env/bin/activate
  pip install -r requirements.txt
```

### Usage ###

```shell
  python run.py
```

And got to http://localhost:5000/ and start the magic
